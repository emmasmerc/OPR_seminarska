# OPR_seminarska
OPIS:
Program predstavlja enostaven knjižnični sistem, kjer lahko dodajamo člane in različna gradiva, ter jih izposojamo.
NAVODILA ZA UPORABA:
Odpri projekt v Visual Studio in zaženi Windows Forms projekt.
Po zagonu prohejta vnesi podatke za knjigo ali člana in klikni ustrezen gumb. S klikom na gumbe prikažeš seznam gradiv ali članov, izberi člana in gradivo ter klikni izposodi gradivo.
